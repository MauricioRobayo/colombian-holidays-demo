## Getting Started

1. Clone the repository and `cd` into it:

```sh
git clone https://github.com/MauricioRobayo/colombian-holidays.git
cd colombian-holidays
```

2. Install dependencies

```sh
npm install
```

3. Start the server

```sh
npm run start
```
